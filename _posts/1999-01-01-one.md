**pre size**在前面一个堆块是空闲的时候有值,用来指示前一个堆块的大小。前面一个堆块在使用时，值始终为 0

size最后一位**PREV_INUSE**用来记录前一个 chunk 块是否被分配,被分配的话这个字段的值为1 

可通过增加 **program break location** 和 **brk** 的方式来增加 **main arena** 的大小

```
#include <stdlib.h>
#include <string.h>

int main(){
        char *a;
        char *b;
        a = malloc(17);
	memcpy(a,"1234567890qwertyu",17);
	b = malloc(25);
	free(a);
        return 0;
}
```
不由得感叹堆设计的精妙,只要是malloc参数除以16的模少于8,就取余数排(形象化说明,一排是16个字节,,否则取余数加一排,如17%16<8,25%16>8,执行完结果
