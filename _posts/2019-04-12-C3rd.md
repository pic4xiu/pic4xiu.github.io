## Impression
> Some small tricks

在学习C语言等较为底层的语言时,考虑到编程效率,我们必须对所写算法进行极致的优化.下面仅仅利用for循环这个例子给大家细致的看一下,for语句怎么用才能达到较为完美的效果.

例如:`计算100~200内的素数`,我们直接上代码

```
#include <stdio.h>
int main()
{
    int j;
    for (int i = 101; i <= 200; i++)
    {
        for (j = 2; j < i / 2; j++)
            if (i % j == 0)
                break;
        if (j == i / 2)
            printf("%d\n", i);
    }
    return 0;
}
```

在这里,我们没有利用数学层面的优化,仅仅利用代码层面暴力求解,且没有引进`math.h`头文件的`sqrt()`函数,只是找遍`i/2`的因数就停止进而输出

很好看懂的一个程序,我们这时候看第二层循环(第一层仅仅是让数字递增,无法优化),我们观察到**for**循环中的**if**语句仅仅是一个`break`操作,我们知道**for**循环中间的条件也是控制循环跳出的,那我们就可以将两者写到一个表达式中进而把算法优化为

```
   for ( j = 2; j < i / 2 && i % j; ++j )
      ;
```

当然我们利用这个的前提是**for**语句中仅含**if**控制程序走向的情况,其他情况还是好好用括号码代码为妙

下面介绍一些**for**语句的简便操作:

改进前:

```
int a[10];
for (int i = 0; i <= 9; i++)
        printf("%d\n", a[i]);
```

从a[0]到a[10]输出

改进后:

```
int a[10]
for (int i=0; i<10; printf("%d ", a[i++])); 
```
